## ❓ 자주 묻는 질문 (FAQ)

### 업로드 관련
**Q: 파일 크기 제한은?**
A: 최대 2GB까지 업로드 가능합니다.

**Q: 지원하는 파일 형식은?**
A: MP4, MOV, AVI, WMV, MKV 형식을 지원합니다.

**Q: 업로드 횟수 제한은?**
A: 비로그인 사용자는 15분당 3회, 로그인 사용자는 15분당 10회까지 업로드 가능합니다.

### 보관 기간
**Q: 업로드한 파일은 얼마나 보관되나요?**
A: 작업 완료 후 7일간 보관되며, 이후 자동으로 삭제됩니다.

**Q: 삭제된 파일을 복구할 수 있나요?**
A: 아니요, 삭제된 파일은 복구할 수 없습니다.

### 계정 관련
**Q: 비밀번호를 잊어버렸어요**
A: 로그인 페이지의 "비밀번호 찾기" 기능을 이용하세요.

**Q: 계정을 삭제하고 싶어요**
A: 마이페이지에서 회원 탈퇴를 진행하세요.

## 🐛 문제 해결

### 업로드 실패
1. 파일 크기 확인 (2GB 이하)
2. 파일 형식 확인 (MP4, MOV, AVI 등)
3. 네트워크 연결 확인
4. 브라우저 새로고침 후 재시도

### 작업이 완료되지 않아요
1. 작업 상태 확인 (저장소 페이지)
2. 에러 메시지 확인
3. 재시도 버튼 클릭
4. 문제가 계속되면 문의

### 다운로드가 안 돼요
1. 작업이 완료되었는지 확인
2. 권한 확인 (본인의 작업인지)
3. 브라우저 설정 확인 (팝업 차단 해제)
