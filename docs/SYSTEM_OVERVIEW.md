# AX2 Caption 시스템 개요

## 📋 시스템 개요

**AX2 Caption**은 AI 기반 자동 자막 생성 웹 애플리케이션입니다. 영상 파일을 업로드하면 AI가 자동으로 음성을 인식하여 다양한 언어로 자막을 생성하는 서비스를 제공합니다. 한국어뿐만 아니라 영어, 일본어, 중국어 등 100개 이상의 언어로 자막을 생성할 수 있습니다.

---

## 🎯 주요 기능

### 1. 자동 자막 생성
- **다국어 음성 인식**: 고정밀 AI 음성 인식 기술로 다양한 언어의 음성을 텍스트로 변환
- **100개 이상 언어 지원**: 한국어, 영어, 일본어, 중국어, 스페인어, 프랑스어, 독일어 등 다양한 언어로 자막 생성
- **언어 자동 감지**: 영상의 음성 언어를 자동으로 감지하여 해당 언어로 자막 생성
- **시간 동기화**: 영상 시간에 맞춰 자막 자동 동기화
- **정확도**: 평균 99% 이상의 정확도

### 2. 자막 편집
- **텍스트 편집**: 자막 텍스트 직접 수정
- **시간 조정**: 자막의 시작 및 종료 시간 미세 조정
- **자막 분할/병합**: 자막 분할 및 병합 기능
- **스타일 설정**: 폰트, 크기, 색상, 배경, 위치 등 자유롭게 설정
- **실시간 미리보기**: 편집한 자막을 영상과 함께 실시간 미리보기

### 3. 파일 관리
- **자동 저장**: 생성된 자막 파일 자동 저장
- **작업 목록 관리**: 모든 작업을 한 곳에서 확인하고 관리
- **검색 기능**: 제목, 날짜 등으로 작업 검색
- **다양한 형식 지원**: SRT, VTT, ASS 등 다양한 자막 파일 형식으로 내보내기
- **자동 삭제**: 처리 완료 후 2시간 뒤 자동 삭제로 개인정보 보호

### 4. 사용자 인증
- **소셜 로그인**: Google, Kakao, Naver 소셜 로그인 지원
- **이메일 회원가입**: 이메일 기반 회원가입 및 로그인
- **세션 관리**: 로그인 상태 관리 및 자동 로그인

### 5. 크레딧 시스템
- **시간 기반 크레딧**: 영상 길이에 따라 크레딧 소모
- **크레딧 계산**: 
  - 기본 자막 생성: 분당 10 크레딧
- **무료 크레딧**: 비로그인 사용자에게 무료 크레딧 제공
- **크레딧 충전**: 로그인 사용자 크레딧 충전 기능

## 💾 데이터 저장 구조

### LocalStorage 키
- `users`: 사용자 데이터 배열
- `currentUser`: 현재 로그인한 사용자 정보
- `isLoggedIn`: 로그인 상태 (true/false)
- `creditBalance`: 로그인 사용자 크레딧 잔액

- `videoJobs`: 비디오 작업 목록
- `currentEditVideoTitle`: 현재 편집 중인 비디오 제목
- `siteLanguage`: 사이트 언어 설정

### 지원 언어 (100개 이상)
- **아시아 언어**: 한국어, 일본어, 중국어(간체/번체), 태국어, 베트남어, 인도네시아어, 말레이어, 힌디어 등
- **유럽 언어**: 영어, 스페인어, 프랑스어, 독일어, 포르투갈어, 이탈리아어, 러시아어, 폴란드어, 네덜란드어, 스웨덴어, 노르웨이어, 덴마크어, 핀란드어, 체코어, 헝가리어, 그리스어, 우크라이나어, 로마니아어 등
- **기타 언어**: 아랍어, 터키어, 히브리어 등

### 언어 선택 기능
- **자동 감지**: 영상의 음성 언어를 자동으로 감지하여 해당 언어로 자막 생성
- **수동 선택**: 사용자가 원하는 언어를 직접 선택하여 자막 생성
- **언어별 정확도**: 각 언어에 최적화된 음성 인식 모델 사용

### 사이트 UI 다국어 지원
- `js/i18n.js`: 사이트 인터페이스 다국어 지원
- `data-i18n` 속성으로 HTML 요소에 번역 키 지정
- LocalStorage에 언어 설정 저장

## 🎬 비디오 처리 워크플로우

1. **업로드**
   - 사용자가 영상 파일 업로드 (MP4, MOV, AVI, 최대 2GB)
   - 파일 검증 및 미리보기

2. **언어 설정**
   - 자막 생성 언어 선택 (100개 이상 언어 지원)
   - 자동 감지 기능으로 영상의 음성 언어 자동 인식

3. **크레딧 확인**
   - 필요한 크레딧 계산
   - 잔액 확인 및 부족 시 경고

4. **처리 시작**
   - 크레딧 예약 (선차감)
   - 서버에 작업 요청 (실제 구현 필요)
   - 진행률 표시

5. **결과 저장**
   - 생성된 자막 데이터 저장
   - 작업 목록에 추가

6. **편집 및 다운로드**
   - 편집 페이지에서 자막 수정
   - SRT, VTT, ASS 형식으로 다운로드

## 🚀 향후 개선 사항

### 현재 제한사항
1. **백엔드 없음**: 현재는 LocalStorage 기반 클라이언트 사이드만 구현
2. **실제 AI 처리 없음**: 실제 음성 인식 API 연동 필요
3. **파일 저장**: 실제 영상 파일 저장 및 스트리밍 기능 필요

### 권장 개선사항
1. **백엔드 서버 구축**: Node.js, Python 등으로 REST API 서버 구축
2. **데이터베이스**: 사용자, 작업, 크레딧 등 데이터 영구 저장
3. **파일 스토리지**: AWS S3, Google Cloud Storage 등 클라우드 스토리지 연동
4. **AI 서비스 연동**: 
   - 다국어 음성 인식: Google Speech-to-Text, AWS Transcribe 등 다양한 언어 지원 음성 인식 API
5. **실시간 처리**: WebSocket을 통한 실시간 진행률 업데이트
6. **결제 시스템**: 크레딧 충전을 위한 결제 게이트웨이 연동
